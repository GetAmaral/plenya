### Conceitos-Chave da Medicina Funcional

#### 1. Ritmo Circadiano
- Produção de melatonina (suprimida por luz azul)
- Cortisol (alto manhã, baixo noite - inverso = estresse crônico)
- Sincronização com exposição solar matinal
- Delayed sleep phase syndrome em adolescentes (atraso 2-3h)

#### 2. Eixo HPA (Hipotálamo-Hipófise-Adrenal)
- Hiperativação por estresse crônico
- Cortisol noturno elevado → fragmentação do sono
- Modulação com adaptógenos (ashwagandha, rhodiola)

#### 3. Eixo Intestino-Cérebro
- Disbiose afeta produção de neurotransmissores (GABA, serotonina)
- Probióticos psicobi óticos (L. plantarum PS128, B. longum 1714)
- Permeabilidade intestinal → neuroinflamação

#### 4. Sono e Metabolismo
- <7h/noite → resistência insulínica, obesidade
- Leptina/grelina alteradas → aumento apetite
- Sono profundo (NREM) essencial para restauração metabólica
- Sono REM crítico para memória e regulação emocional

#### 5. Deficiências Nutricionais Frequentes
- **Magnésio:** relaxante muscular, GABAérgico (400-600mg/noite)
- **Ferritina:** <75 ng/ml associada a síndrome pernas inquietas
- **Vitamina D:** receptores no cérebro regulam sono (meta >40 ng/ml)
- **Complexo B:** produção de serotonina/melatonina
- **Ômega-3 EPA/DHA:** anti-inflamatório neural (1-2g/dia)

#### 6. Protocolos Suplementares Recorrentes
- **Sleep Cocktail:** Magnésio treonato + Glicina 3g + L-teanina 200mg
- **Ansiedade:** Ashwagandha 300-600mg + Magnésio + Fosfatidilserina
- **Ritmo Circadiano:** Melatonina 0.5-3mg (temporário) + Exposição solar
- **Anti-inflamatório:** Ômega-3 2g/dia + Vitamina D + NAC

#### 7. Higiene do Sono (Universal)
- Escuridão total (cortinas blackout, sem LEDs)
- Temperatura 18-20°C
- Bloqueio de luz azul 2-3h antes de dormir
- Rotina consistente (mesmos horários inclusive finais de semana)
- Evitar cafeína após 14h (meia-vida 5-6h)

## Aspectos Técnicos da Implementação

### Stack Utilizado

- **Backend:** Go 1.25 + Fiber v2 + PostgreSQL 17
- **Script:** Python 3 + requests library
- **API:** RESTful (PUT /api/v1/score-items/{id})
- **Banco:** Docker Compose managed PostgreSQL

### Correções Realizadas

#### Problema 1: Campo do Token
- **Erro:** KeyError: 'token'
- **Causa:** API retorna `accessToken`, não `token`
- **Solução:** Ajuste em `login()` function

#### Problema 2: Campos Não Salvos
- **Erro:** Apenas `conduct` sendo salvo, `clinical_relevance` e `patient_explanation` NULL
- **Causa:** Python script usando snake_case, mas DTO Go espera camelCase
- **Solução:**
  ```python
  # Errado
  data = {"clinical_relevance": ..., "patient_explanation": ...}

  # Correto
  data = {"clinicalRelevance": ..., "patientExplanation": ...}
  ```

### Estrutura do DTO (Go Backend)

```go
type UpdateScoreItemDTO struct {
    ClinicalRelevance  *string `json:"clinicalRelevance,omitempty"`
    PatientExplanation *string `json:"patientExplanation,omitempty"`
    Conduct            *string `json:"conduct,omitempty"`
    LastReview         *time.Time `json:"lastReview,omitempty"`
}
```

- **Lógica:** Cada item vinculado a 3 artigos de Ritmo Circadiano
