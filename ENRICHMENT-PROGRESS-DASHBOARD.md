## Tecnologia e Metodologia

### Stack Validado

- **Backend:** Go Fiber + PostgreSQL 17
- **Scripts:** Python 3 + requests
- **Autenticação:** JWT Bearer Token
- **Validação:** Query PostgreSQL
- **Versionamento:** Git + relatórios Markdown

### Processo Otimizado

1. Login automático via API
2. Iteração sobre lista de items
3. Geração de conteúdo baseado em template + conhecimento MFI
4. PUT request para cada item
5. Delay de 0.5s entre requisições
6. Retry automático em caso de falhas
7. Validação no banco de dados
8. Geração de relatórios JSON + Markdown

### Tempo de Processamento

- **20 items:** 8-10 minutos (incluindo retry)
- **Taxa de sucesso:** 85-100% (primeira tentativa)
- **Taxa de sucesso após retry:** 100%
