## COMANDOS ÚTEIS

### Verificar item específico
```bash
docker compose exec -T db psql -U plenya_user -d plenya_db -c "
SELECT name, clinical_relevance, patient_explanation, conduct
FROM score_items
WHERE name ILIKE '%termo%';
"
```

### Listar artigos de um item
```bash
docker compose exec -T db psql -U plenya_user -d plenya_db -c "
SELECT a.title
FROM articles a
INNER JOIN article_score_items asi ON a.id = asi.article_id
INNER JOIN score_items si ON asi.score_item_id = si.id
WHERE si.name = 'Nome do Item';
"
```

### Buscar items sem enriquecimento
```bash
docker compose exec -T db psql -U plenya_user -d plenya_db -c "
SELECT name FROM score_items
WHERE (clinical_relevance IS NULL OR clinical_relevance = '')
AND group_name = 'Alimentação';
"
```
